### STEP 1: HTML REVIEW

```
<!doctype html>
<html>
  <head></head>
  <body></body>
</html>
```

- HTML is a “text file”
- HTML is composed of HTML Tags
- HTML tags can be nested inside of each other

Image Tags:

```
<img src="/images/boat.jpg" />
```

Header Tags:
